  2. Enter the code from the map

Red Access Cards
  1. Locate a Red Access Card from a random loot drop
  2. Go to a marked map location to access the bunker

Easter egg is currently unsolved

  1. Locate the computers around the map

Secret Station
  1. Travel to the marked #1 building on the map
  2. Interact with the keypad and enter "2179"
  3. Interact with the computer
  4. Write down each photo tag (and differential if there)
  5. Note the Roman Numeral under the painting, then add/subtract the differential to decipher the code
  6. Interact with the computer again to enter the top halves, then bottom halves of the code
  7. Travel to the marked #2 tunnel on the map
  8. Interact with the computer, get on the subway
